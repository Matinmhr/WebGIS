# Belgium Air Quality Analysis — GIS LAB Group 16

## Overview
This project presents a spatial analysis of air quality across Belgium, focusing on three key pollutants: NO₂, PM2.5 and PM10. The study analyzes data from 2021 and 2023 and compare the result of the two years.

The project was developed as part of the Lab of "Geographic Information Systems" course at Politecnico di Milano (A.A. 2025/26).
